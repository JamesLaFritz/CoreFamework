    FolderPathAttribute

    Description

    Displays a text field with a folder button to the right, with which you can select a folder path as the string value. If you pass false to this attribute this will use the absolute path to the folder. If you pass true to this attribute this will use the path relative to the Project.

    To make it so the field can not be change outside the Class make it a private field and add the SerializeField tag.

    [readOnly, SerializeField] public int healthPoints;
[readOnly] [SerializeField] public int healthPoints;
    Examples
    
 [FolderPath(false)] public string absolutePath;
 
    
 [FolderPath(true)] public string relativePath;
